# Dashboard de Farmacia

Dashboard desarrollado con Flask y MySQL para visualizar indicadores de una farmacia.

## Tecnologías

- Python
- Flask
- MySQL
- HTML
- CSS

## Instalación

1. Clonar el repositorio

git clone https://github.com/lucascapocasa1/TFI---Gestion-de-Datos---UNAB.git

2. Instalar dependencias

pip install -r requirements.txt

3. Importar la base de datos

Abrir MySQL Workbench y ejecutar:

SOURCE database/farmacia_tfi.sql;

4. Ejecutar la aplicación

python app.py

5. Abrir en el navegador

http://localhost:5000
